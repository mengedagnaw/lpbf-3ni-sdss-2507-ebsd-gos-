# LPBF 3 wt.% Ni-Modified SDSS 2507 EBSD GOS Analysis

This repository contains MATLAB/MTEX scripts and figures for EBSD-based grain orientation spread (GOS) analysis of laser powder bed fusion (LPBF)-fabricated 3 wt.% Ni-modified super duplex stainless steel 2507.

## Scope

The repository supports microstructural misorientation analysis using GOS maps and GOS number-fraction histograms across the investigated processing and post-processing conditions.

The processing conditions included in the uploaded figure are:

- AS
- SR400
- SR450
- SR500
- SR550
- SA1100
